# Jackaroo-Game
Implemented Jackaroo game full back-end logic using OOP  with programming language: Java and integrated it with the front-end using JavaFX
Jackaroo Game – JavaFX Edition

A fully interactive Jackaroo board game built using JavaFX, featuring a custom-designed Ancient Egyptian theme, dynamic UI, and integrated background music.

📌 Overview
This project is a complete implementation of the Jackaroo card & board game, combining:
 Custom UI (JavaFX)
 Game Logic (Backend Engine)
 Background Music System
 Interactive Views & Navigation
The game delivers a visually immersive experience inspired by Ancient Egypt, with pharaoh-styled cards and themed interfaces.

✨ Features
 Gameplay
Full Jackaroo game logic implementation

Marble movement and interaction
Turn-based gameplay
